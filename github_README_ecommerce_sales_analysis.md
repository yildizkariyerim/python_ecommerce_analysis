# E-Commerce Sales Analysis Project

## Project Overview

This project analyzes a sample e-commerce sales dataset using Python and Pandas.

The goal of the project is to understand order behavior, customer segments, revenue distribution, and business insights.

## Dataset

The dataset includes 100 sample e-commerce orders with the following columns:

- order id
- order date
- customer name
- city
- category
- product
- membership type
- payment method
- product price
- quantity
- discount rate
- total amount
- shipping duration
- order status

## Tools Used

- Python
- Pandas
- Google Colab

## Analysis Steps

1. Loaded CSV file into Google Colab
2. Explored the dataset with `head()`, `shape`, `info()`, and `describe()`
3. Calculated summary statistics
4. Filtered high-value orders
5. Analyzed customer segments by membership type
6. Compared revenue by category and city
7. Generated business insights

## Key Insights

- The average order amount is approximately 1759 TL, while the median is approximately 1173 TL. This suggests that some high-value orders increase the average.
- Completed orders generated approximately 158272 TL total revenue.
- The top revenue-generating category is Elektronik.
- Gold members have a higher average order amount than Standard members.
- Order values vary significantly, so customer segmentation can support better marketing decisions.

## Business Recommendations

- Focus retention campaigns on high-value customers.
- Analyze top-performing categories for growth opportunities.
- Create targeted campaigns for Gold and Silver members.
- Monitor high-value orders and shipping experience closely.
